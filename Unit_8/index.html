                let a = 0; 
                do
                     console.log('do while work');

                let b = 0;
                while (b < 0) {
                    console.log('while work');
                }
изучите вывод. Как видите, цикл do while срабатывает минимум один раз в любом случае. Т.е.вначале идет срабатывание, а потом проверка.
